# 📻 Morse Code Game

Aprende **Código Morse** jugando. Esta es una aplicación web interactiva diseñada para practicar, decodificar y codificar mensajes mediante un desafío cronometrado que mide tu velocidad y precisión. 

### 🎮 [¡Haz clic aquí para jugar ahora!](https://francojaq2.github.io/Morse-Game/)

## ✨ Características Principales
* **Tabla Interactiva:** Explora el alfabeto Morse con retroalimentación de audio en tiempo real.
* **Desafío Cronometrado:** Pon a prueba tus reflejos y tu memoria bajo presión.
* **Diseño Moderno:** Interfaz limpia y progresión clara para dominar el Morse de forma práctica y divertida.

## 🛠️ Tecnologías Utilizadas
* HTML5
* CSS3
* Vanilla JavaScript
